# Akash Yadav - Portfolio

A production-level, high-performance portfolio website built with React, Vite, and Framer Motion.

## Features
- **Premium Dark Aesthetics**: Sophisticated color palette and modern typography (Outfit & Inter).
- **Dynamic Animations**: Smooth scroll-triggered animations and glassmorphism effects.
- **Production Data**: Populated with real metadata from your resume (Mintoak, Swipe, HDFC Bank, Axis Bank).
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Interactive Stack**: Categorized technical skills with visual feedback.

## Tech Stack
- **Frontend**: React (Hooks)
- **Styling**: Vanilla CSS with CSS Variables
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
